---
description: How to run the backend server
---

1. Open a new terminal.
2. Navigate to the backend directory:
   ```powershell
   cd nexus-backend
   ```
3. Install dependencies (if not already installed):
   ```powershell
   npm install
   ```
4. Start the development server (with auto-restart):
   ```powershell
    npm run dev
   ```
   OR standard start:
   ```powershell
   npm run start
   ```
5. The server should be running on http://localhost:3001.
